# IED Barcelona · AI Workshop Landing Page

Editorial-style landing page for the workshop:
**Building Web Apps Without Coding**

## Stack
- React + TypeScript + Vite
- [motion.dev](https://motion.dev/) via `motion`
- Typography: **Fraunces** (serif) + **Inter** (sans-serif)

## Design Notes
- Light visual system
- Accent color sampled from IED web styles: `#ff3232`
- Clean editorial rhythm with generous spacing and typography-led hierarchy

## Content Sources (2024–2026)
- IED Business Design page (program framing, project-based learning)
- IED Barcelona page (context around innovation ecosystem)
- GitHub research on Copilot productivity (55% faster task completion)
- 2025 reports on YC W25 AI-generated codebase trend
- Publicly visible LinkedIn profile snippets for Jose Eduardo Saura Oller
